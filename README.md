# Projet Reuninou (WebApp)

*Atelier du 21/03/2022 au 30/03/2022*

*[Lire le README-GLOBAL du projet](./README-GLOBAL.md)*

## Installation

Dans le dossier `/webapp`

```sh
npm install
```

### Compile and Hot-Reload for Development

Dans le dossier `/webapp`

```sh
npm run dev
```

### Compile and Minify for Production

Dans le dossier `/webapp`

```sh
npm run build
```
